# Менің бірінші ЖИ-Ассистентім 🤖

**Практикалық жұмыс №1 — ЖИ веб-әзірлеуде**

## Жоба туралы

Бұл жоба ЖИ-құралдарын нақты веб-жобада қолдануды көрсетеді. Бетте ЖИ-ді веб-әзірлеуде пайдалану мүмкіндіктері шолынады және тікелей чат интеграциясы берілген.

**Жоба мынаны қамтиды:**
- Midjourney, Google Gemini, Suno AI туралы карточкалар
- Адаптивті дизайн (мобильді + десктоп)
- Қараңғы/жарық тақырып ауыстыру
- Claude API интеграциясы (чат)
- Карточкалардың айналдыру анимациясы
- Сұраныстар санауышы

## Қалай іске қосу керек

1. Репозиторийді клондаңыз:
   ```bash
   git clone https://github.com/username/ai-assistant-project.git
   cd ai-assistant-project
   ```

2. `script.js` файлындағы `YOUR_API_KEY_HERE` орнына өз Claude API кілтіңізді қойыңыз:
   ```js
   const API_KEY = 'sk-ant-...'; // claude.ai сайтынан алыңыз
   ```

3. `index.html` файлын Chrome немесе Firefox браузерінде ашыңыз.

> **Ескерту:** API кілтінсіз чат «Демо режимде» жұмыс жасайды.

## Файлдар құрылымы

```
/

```

## Пайдаланылған құралдар

| Құрал | Мақсат |
|-------|--------|
| **Claude AI** | Карточка мәтіндері мен кодқа кеңес |
| **Google Fonts** | Syne + Space Mono қаріптері |
| **Claude API** | Чат интеграциясы |
| **CSS Grid / Flexbox** | Адаптивті орналасу |
| **Intersection Observer API** | Айналдыру анимациясы |

## Мен не үйрендім

- **CSS Custom Properties (айнымалылар)** арқылы тақырыпты динамикалық ауыстыруды үйрендім
- **Intersection Observer API** — JavaScript-те жаңа веб API-мен алғаш танысқаным болды
- `async/await` синтаксисін `fetch()` арқылы API шақыруда қолдануды үйрендім
- CSS `grid-template-columns: repeat(3, 1fr)` + медиа сұраныстары арқылы адаптивті карточкалар жасадым
- ЖИ-мен тиімді жұмыс жасауды: нақты сұрақ қою маңызды, «бәрін жаз» деген сұрақтан гөрі «fetch() мысал көрсет» деген тиімді
